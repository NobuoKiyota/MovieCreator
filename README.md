# MovieCreator - Generative Video Generator

ブラウザ上で動作する、ネオン・サイバーパンク風の生成映像・効果映像クリエイターです。
多数のレイヤー、LFOオートメーション、キーフレームタイムライン、新ジェネレーター6種（炎、雪の結晶、3Dパーティクル等）や各種ポストプロセスエフェクト（万華鏡、色収差等）を駆使し、リアルタイムに変異・モジュレーションする美しい映像を構築・書き出し（MP4/透過WebM）できます。

---

## 他のPCで実行する方法

本アプリケーションはウェブブラウザ上で動作するため、他PCへの導入も非常に簡単です。用途に合わせて以下の3つの方法から選択できます。

### 方法1：ホスティングしてURLでアクセスする（最もおすすめ・インストール不要）
このプロジェクトは完全にフロントエンドのJavaScriptだけで完結しているため、GitHub Pagesなどの静的ホスティングサービスにデプロイすることで、**他のPCからは指定のURLにブラウザでアクセスするだけでインストールなしで即座に動作します**。
（※お手元のGitHubリポジトリを活用してデプロイ可能です。ご希望の場合は自動デプロイ設定も構築できます）

### 方法2：ローカル実行（Node.jsを使用）
Node.js環境を利用してローカルサーバーを起動し実行する方法です。

#### 必要なもの
- **[Node.js (LTS推奨)](https://nodejs.org/)** のインストール

#### 手順
1. アプリケーションのフォルダ（プロジェクト一式）を他PCにコピーします。
2. フォルダ内でターミナル（Command Prompt / PowerShell）を開き、以下のコマンドを実行して依存ライブラリをインストールします。
   ```bash
   npm install
   ```
3. インストール完了後、フォルダ内の `run_app.bat` をダブルクリックするか、以下のコマンドを実行します。
   ```bash
   npm run dev
   ```
4. 画面に表示されるローカルURL（例: `http://localhost:5173/` など）にブラウザでアクセスします。

### 方法3：ビルド成果物 (dist フォルダ) + 簡易Webサーバーによる実行
Node.jsをインストールせず、ビルド済みの成果物（HTML/JS/CSS）を簡易サーバーで動かす方法です。
ブラウザのセキュリティ制約（CORSポリシー）により、HTMLファイルを直接ダブルクリック (`file://`) するだけでは動作しないため、軽量なサーバーが必要です。

#### 簡易起動の手順
1. 本PCで一度 `npm run build` を実行して作成された `dist/` フォルダを、他PCにコピーします。
2. 他PCに Python がインストールされている場合、`dist/` フォルダ内で以下のコマンドを実行するだけでWebサーバーが立ち上がります。
   ```bash
   python -m http.server 8000
   ```
   その後、ブラウザで `http://localhost:8000` にアクセスします。
3. （ご希望であれば、ダブルクリックするだけでローカルWebサーバーを起動できる超軽量な単体サーバープログラムを実行バッチと同梱して配布用パッケージを作成することも可能です）
